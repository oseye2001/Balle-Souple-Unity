# Simulation d’une balle souple (Unity)

Projet de simulation physique en Unity
Modèle masse–ressort–amortisseur
Rebond dynamique lié à l’impact
Interactions physiques avec métal et verre

## Scripts principaux

- BallDeformer.cs
- BallKickOnFirstContact.cs
- CameraZoom.cs
- GlassShake.cs

## Physique utilisée

F = -kx - cv

- k : raideur (stiffness)
- c : amortissement (damping)
- x : compression

Ouvrir la scène principale :
